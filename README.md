# Smart Cashier

Aplikasi kasir berbasis Java menggunakan NetBeans dan MySQL.

## Features
- Login
- CRUD Produk
- CRUD Transaksi
- Manajemen Kasir

## Teknologi
- Java
- NetBeans
- MySQL

## Cara Menjalankan
1. Clone repository
2. Import project ke NetBeans
3. Jalankan database MySQL
4. Run project
